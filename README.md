# Java婚纱影楼管理系统 java705

## 项目概述

Java婚纱影楼管理系统是一款基于B/S结构的Web应用，采用前后端分离的技术架构，后端运用SpringBoot、SSM框架，前端采用Vue技术，实现了影楼业务的全方位管理。

## 技术栈

### 后端技术

- Java
- SpringBoot
- MySQL
- Spring
- SpringMVC
- MyBatis

### 前端技术

- Vue
- CSS
- HTML5
- JavaScript

### 开发及运行环境

- 系统环境：Windows/Mac
- 开发环境：JDK 1.8、Maven、MySQL、Tomcat
- 开发工具：Eclipse/MyEclipse/IDEA
- 数据库工具：Navicat 11
- 服务器：Tomcat 9.0
- 数据库：MySQL 5.7（推荐）

## 功能模块

系统主要功能模块涵盖了影楼业务流程，具体包括但不限于：

- 影楼预约管理
- 客户信息管理
- 婚纱礼服管理
- 摄影师分配
- 订单处理
- 财务报表

## 系统角色

- 系统管理员
- 前台接待
-摄影师
- 化妆师
- 财务人员

## 部署步骤

1. 环境搭建：配置Java、MySQL、Tomcat等开发环境。
2. 数据库部署：使用Navicat导入数据库文件。
3. 项目编译：使用Maven进行项目编译。
4. 部署应用：将编译后的代码部署到Tomcat服务器上。
